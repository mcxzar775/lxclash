# 龙行科技 v2.3 独立动态背景 UI

本版本在完整 Clash Party/Mihomo 功能底座上重新开发龙行科技独立界面，没有删除原功能。

## 保留功能

- 系统代理与 TUN
- 节点、代理组和延迟测试
- 订阅、配置文件和专属授权订阅
- 连接、日志、规则、DNS、嗅探
- 覆写、外部资源、Sub-Store
- 网络拓扑、流量统计和内核管理
- 托盘、更新、备份、快捷键及其他原设置
- 龙行科技授权码登录和退出授权

## 新界面

- 独立固定侧栏
- 全功能中文导航
- 顶部模式控制区
- 系统代理、TUN、订阅、节点快捷控制区
- 半透明玻璃面板
- MP4 循环动态背景
- 窄屏自适应

背景文件：`src/renderer/src/assets/longxing-bg.mp4`
